很明显gdm3是我的显示管理器

那么关桌面应该只要关掉gdm3就行了,我又问了 phind,它说:

  1. 打开终端
  2. 输入命令 sudo systemctl stop gdm3.service 停止gdm3服务
  3. 如果想要禁止gdm3服务在开机时启动，可以输入 sudo systemctl disable gdm3.service
  4. 如果想要重新开启gdm3服务，可以输入 sudo systemctl start gdm3.service

结果

我的电脑桌面成功变成了闪烁的下划线,图形界面关闭,所以说授人以渔嘛,关闭图形界面要看你用的是什么显示服务,上面的方式应该可以解决大多数的问题
